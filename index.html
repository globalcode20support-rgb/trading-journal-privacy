📄 Política de Privacidad

Última actualización: 16 de marzo de 2026

Resumen rápido:
Trading Journal es una aplicación privada. Todos los datos se almacenan exclusivamente en el dispositivo del usuario. No recopilamos, transmitimos ni almacenamos datos en servidores externos.

1. Introducción

Esta Política de Privacidad describe cómo la aplicación Trading Journal gestiona la información del usuario.

Trading Journal ha sido diseñada con la privacidad como principio fundamental. Los datos de trading pueden ser extremadamente sensibles, por lo que la aplicación funciona completamente de forma local en el dispositivo del usuario.

2. Datos que NO recopilamos

Trading Journal no recopila ni transmite:

  • Información personal (nombre, email, teléfono)
  • Datos de operaciones de trading
  • Resultados o estrategias
  • Imágenes o capturas de análisis
  • Ubicación del usuario
  • Datos de uso de la aplicación
  • Información del dispositivo
  • Datos analíticos o publicitarios

3. Almacenamiento local

Todos los datos se almacenan exclusivamente en el dispositivo del usuario utilizando tecnologías locales del navegador:

  • LocalStorage para configuración y operaciones
  • IndexedDB para imágenes y capturas

Esto significa que los datos nunca abandonan el dispositivo del usuario.

4. Permisos de la aplicación

La aplicación puede solicitar permisos opcionales:

  • Cámara (para tomar fotos de análisis)
  • Galería o almacenamiento (para seleccionar imágenes)
  • Notificaciones locales

Todos estos permisos son opcionales y pueden revocarse en cualquier momento desde la configuración del dispositivo.

5. Servicios de terceros

Trading Journal no integra servicios de rastreo ni análisis como:

  • Google Analytics
  • Firebase Analytics
  • Facebook SDK
  • Redes publicitarias

La aplicación puede permitir conexiones opcionales a plataformas externas como TradingView o Investing.com si el usuario decide utilizar dichas funcionalidades.

6. Seguridad de los datos

Debido a que los datos permanecen en el dispositivo del usuario, recomendamos:

  • Proteger el dispositivo con contraseña o biometría
  • Mantener el sistema operativo actualizado
  • Realizar copias de seguridad periódicas
  • Eliminar los datos antes de vender o transferir el dispositivo

7. Derechos del usuario

El usuario tiene control total sobre sus datos y puede:

  • Acceder a todos los datos almacenados
  • Editar cualquier información
  • Exportar sus datos
  • Eliminar completamente todos los datos

8. Cumplimiento RGPD

Aunque Trading Journal no recopila datos personales, se respetan los principios del Reglamento General de Protección de Datos (RGPD).

9. Menores de edad

La aplicación está destinada a usuarios mayores de 18 años debido a la naturaleza del trading financiero.

10. Cambios en esta política

Esta política puede actualizarse ocasionalmente. La fecha de actualización siempre se indicará en esta página.
